from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# Cargar el dataset Iris
iris = load_iris()
X = iris.data  # características: largo/peso del sépalo y pétalo
y = iris.target  # clases: 0=setosa, 1=versicolor, 2=virginica

# Dividir en conjunto de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Crear y entrenar el clasificador KNN
model = KNeighborsClassifier(n_neighbors=3)
model.fit(X_train, y_train)

# Hacer predicciones
y_pred = model.predict(X_test)

# Evaluar precisión
accuracy = accuracy_score(y_test, y_pred)
print(f"Precisión del modelo: {{'{'}}accuracy:.2f{{'}'}}")

# Ejemplo de predicción con nueva flor
nueva_flor = [{{'['}}5.1, 3.5, 1.4, 0.2{{']'}}]  # se espera que sea setosa
prediccion = model.predict(nueva_flor)
print("Tipo de flor predicha:", iris.target_names[prediccion[0]]{{ '}' }})

Precisión del modelo: 1.00

Queremos que la computadora aprenda a reconocer el tipo de flor (del conjunto Iris) a partir de algunas medidas, como:
  • Largo y ancho del sépalo
  • Largo y ancho del pétalo
A cada flor se le conoce su tipo (por ejemplo: Setosa, Versicolor o Virginica), y el objetivo es que el programa aprenda a distinguir entre ellas según las características.

🔍 ¿Cómo lo logramos?
  • Le damos datos reales al programa usando el dataset Iris.
  • Dividimos los datos en dos grupos: uno para entrenar y otro para probar.
  • Elegimos un modelo simple: K-Nearest Neighbors (KNN).
  • Entrenamos el modelo con el grupo de entrenamiento.
  • Probamos su precisión con datos nuevos.
  • Hacemos una predicción con una flor nueva.
🧠 ¿Qué significa esto?
Hemos creado un modelo que aprende de ejemplos. Es como si le mostraras a alguien muchas flores con su nombre y luego le dieras una flor nueva para que te diga de qué tipo es. Eso es aprendizaje automático supervisado, un tipo de inteligencia artificial.
