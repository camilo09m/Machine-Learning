¿Qué son los Árboles de Decisión?

Un Árboles de Decisiones (o decision tree, en inglés) es un modelo de Machine Learning utilizado principalmente para tareas de clasificación y regresión. Se representa como una estructura en forma de árbol, donde cada nodo interno representa una decisión basada en una característica (o atributo) de los datos, y cada rama representa el resultado de esa decisión. Finalmente, cada hoja del árbol representa una predicción o una etiqueta de clase.
